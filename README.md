# Sports Scouting Platform

A web-based scouting platform that allows coaches and analysts to view detailed reports on Premier League teams. Designed to streamline opposition analysis by replacing manual scouting with structured, digital data.

## Features
- Role-based login for coaches and analysts
- Dashboard showing 10 Premier League teams
- Detailed scouting reports including stats, tactics, and player insights
- Clean, responsive UI for easy navigation

## Tech Stack
- Frontend: React.js, Material UI
- Backend: Node.js (Express) / Django
- Database: PostgreSQL
